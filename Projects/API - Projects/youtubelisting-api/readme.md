# 🎥 Video Hub (YouTube UI Clone)

A clean YouTube-style video listing interface built using a public Videos API.

## 🚀 Features
- Fetch videos from API
- Responsive grid layout
- Video cards (thumbnail, title, channel, views)
- 🔍 Search functionality (dynamic query)
- Clean and modern UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (Fetch API)

## 📡 API Used
https://api.freeapi.app/api/v1/public/youtube/videos

## ⚙️ Run Locally
1. Clone the repo
2. Open with Live Server (VS Code recommended)
